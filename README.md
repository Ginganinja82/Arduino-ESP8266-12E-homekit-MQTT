# Arduino-ESP8266-12E-homekit-MQTT


ESP8266 model 12E used to communicate to APPLE HOMEKIT via MQTT protocall and HOMEBRIDGE

# Requirements: 
* _MQTT service running (..or accesable from your local network)   
            ![](https://raw.githubusercontent.com/mqttjs/MQTT.js/137ee0e3940c1f01049a30248c70f24dc6e6f829/MQTT.js.png)
* _Apple HOMEBRIDGE (or HAP-Node.js) installation        
            ![](http://photos2.appleinsider.com/gallery/15192-11300-HomeKit-l-l.jpg)
* _Arduino IDE with updated boards  
            ![](https://www.arduino.cc/en/pub/skins/arduinoWide/img/ArduinoAPP-01.svg)
* _ESP8266-12E     
            ![](http://potentiallabs.com/cart/image/cache/catalog/ESP8266/ESP12E-500x416.jpg)
* _USB to Serial programmer \ Arduino UNO to use as ISP   
            ![](https://cdn.instructables.com/FRU/7M4T/HFD1F0NG/FRU7M4THFD1F0NG.MEDIUM.jpg)
